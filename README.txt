*** Fooda Take home Coding Challenge ***

This program is for TrackingRewardsPoints for Customer Orders. This project when run will provide an output that has list of all customers, their total points and points gained per order.

The program will take a json file as input defined in the location:
src/main/resources/

In order to input a custom json file of your choice, please change add the new json file in above mentioned location and also change the file input path in ReadJSONfile.java 

You replace the path of json file in above file at line 26 with
src/main/resources/<<new_file.json>>

Once changed, refresh the file and run the program and it will return the output.
